# Getting Started

This section walks you through installing DEVON and downloading your first model.
By the end you will have a working CLI, a model in your local vault, and an
understanding of the core workflow.

---

## Where to Go

| Page | Description |
|---|---|
| [Installation](installation.md) | Clone the repository, install dependencies with Poetry, and verify the CLI |
| [Tutorial: First Model](first-model.md) | Search HuggingFace, download a model, inspect your vault, and export paths for KITT |

---

## Prerequisites at a Glance

| Requirement | Minimum Version |
|---|---|
| Python | 3.10+ |
| Poetry | 1.7+ |
| git | any recent version |

!!! note
    Unlike KITT, DEVON does not require Docker. It runs natively on your system
    using only Python and Poetry.
