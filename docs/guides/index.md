# Guides

How-to guides for common tasks with DEVON. Each guide focuses on a
specific workflow and provides concrete commands you can copy and run.

## Available Guides

### [Searching for Models](searching.md)

Find models on HuggingFace using keyword search and filters for provider,
parameter count, file size, format, task type, and license.

### [Downloading Models](downloading.md)

Download models by URL or model ID, select specific files with include
patterns, and manage disk space.

### [Managing Local Models](managing.md)

List, inspect, and remove downloaded models. Monitor storage usage and
clean up unused files.

### [KITT Integration](kitt-integration.md)

Export your local model library for use with KITT, the inference engine
testing suite.

### [Configuration](configuration.md)

Customize DEVON's behavior with a YAML configuration file. Set storage
paths, search defaults, download options, and display preferences.

---

If you are new to DEVON, start with the
[Getting Started](../getting-started/index.md) guide first, then return
here when you need help with a specific task.
